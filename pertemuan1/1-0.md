Halo! Pilihan yang keren banget buat mulai dari Linux. Di dunia cyber security, Linux itu ibarat "rumah" sendiri—hampir semua alat pentesting, server, dan infrastruktur keamanan berjalan di atas ekosistem ini.



Karena kamu sudah punya dasar jaringan, transisi ke Linux akan terasa jauh lebih masuk akal. Berikut adalah roadmap belajar yang terstruktur, mulai dari nol sampai ke tahap praktik keamanan.



Phase 1: Linux Essentials (Fondasi Kuat)

Jangan langsung lompat ke hacking tools. Kamu harus nyaman dulu di dalam terminal (Command Line Interface - CLI).



Navigasi \& File System: Pahami hierarki direktori Linux (kenapa /etc penting, di mana log disimpan dalam /var/log).



User \& Permissions: Belajar konsep root, sudo, dan cara kerja permissions (rwx). Ini krusial karena banyak exploit terjadi karena kesalahan izin file.



Package Management: Cara install aplikasi lewat terminal menggunakan apt, pacman, atau dnf.



Text Editors: Minimal kuasai Nano atau Vim untuk edit file konfigurasi langsung dari terminal.



Phase 2: Linux System Administration

Di tahap ini, kamu belajar mengelola sistem seperti seorang admin profesional.



Service \& Process Management: Belajar cara menjalankan, mematikan, dan memantau layanan (misal: Apache, MySQL, SSH) menggunakan systemctl dan top/htop.



Networking in Linux: Karena kamu sudah belajar dasar jaringan, sekarang praktikkannya di Linux. Coba perintah seperti ip addr, netstat, nmap, dan cara konfigurasi firewall sederhana dengan iptables atau ufw.



Log Analysis: Belajar cara membaca log sistem untuk mendeteksi aktivitas mencurigakan.



Phase 3: Scripting \& Automation

Seorang cyber security enthusiast yang hebat tidak melakukan semuanya secara manual.



Bash Scripting: Belajar cara membuat tools sederhana untuk otomatisasi tugas (misal: auto-backup atau simple port scanner).



Python for Security: Python adalah bahasa paling populer di komunitas ini. Mulailah belajar dasar Python untuk berinteraksi dengan sistem dan jaringan.



Phase 4: Security Hardening (Defensive)

Sebelum menyerang, pelajari cara bertahan.



SSH Hardening: Cara mengamankan akses jarak jauh (ganti port, pakai SSH Key, matikan root login).



User Auditing: Memastikan tidak ada user yang punya hak akses berlebih.



Kernel Security: Mengenal apa itu Kernel dan bagaimana menjaganya tetap up-to-date untuk menghindari kerentanan.



Phase 5: Offensive Linux \& Practice (Practical)

Sekarang waktunya mencoba alat-alat yang ada di distro khusus seperti Kali Linux atau Parrot OS.



Reconnaissance: Menggunakan alat seperti Nmap untuk memetakan jaringan.



Vulnerability Assessment: Belajar cara mencari kelemahan pada layanan yang berjalan di Linux.



Capture The Flag (CTF): Mulailah bermain di platform seperti TryHackMe atau Hack The Box. Mereka punya "Linux Path" yang sangat bagus untuk pemula.



Tips Tambahan:

Jangan instal Linux sebagai sistem operasi utama dulu kalau ragu. Gunakan VirtualBox atau VMware biar kamu bebas "merusak" sistemnya tanpa takut data pribadimu hilang.

🛡️ Bootcamp Roadmap: The Cyber Path

🟦 Level 1: The Linux Architect (Newbie)

Fokus: Menjadikan terminal sebagai "rumah kedua". Kamu tidak bisa meretas sistem jika kamu tidak tahu cara kerjanya.



Linux Fundamental: Instalasi (Distro bebas, tapi coba Debian-based), Navigasi CLI (cd, ls, mkdir), dan manipulasi file (cat, grep, sed, awk).



User Management: Memahami sudo, chown, chmod, dan manajemen grup.



Networking Basics (Linux Side): Belajar cara kerja IP Address, DNS, dan DHCP langsung dari terminal menggunakan ip, dig, dan nmcli.



Services \& Ports: Cara menjalankan dan mengamankan web server (Apache/Nginx) serta SSH.



🏁 Milestone Project: "The Hardened Server" — Konfigurasi server Linux yang aman, pasang firewall (UFW), dan matikan semua port yang tidak perlu.



🟨 Level 2: The Digital Scout (Beginner)

Fokus: Memahami cara kerja internet dan mulai melakukan pengintaian (reconnaissance).



Advanced Networking: Model OSI, TCP/UDP, Subnetting, dan sniffing paket menggunakan Wireshark.



Web Vulnerabilities (OWASP Top 10): Belajar dasar SQL Injection, Cross-Site Scripting (XSS), dan Broken Authentication.



Footprinting \& Scanning: Menguasai Nmap untuk memetakan jaringan dan Dirsearch untuk mencari direktori tersembunyi.



Introduction to Tools: Mulai berkenalan dengan Burp Suite (Community Edition) untuk intercept trafik web.



🏁 Milestone Project: "Vulnerability Report" — Cari kelemahan di lab lokal (misal: mesin Metasploitable) dan buat dokumentasi temuannya.



🟧 Level 3: The Ghost Breaker (Advanced)

Fokus: Eksploitasi, peretasan sistem, dan menjaga akses.



System Exploitation: Menggunakan Metasploit Framework untuk menyerang layanan yang rentan.



Privilege Escalation: Belajar cara naik dari user biasa menjadi root (Linux) atau System (Windows).



Wireless Hacking: Memahami protokol keamanan WiFi (WPA2/WPA3) dan cara kerjanya.



Binary \& Scripting: Belajar Python atau Bash untuk membuat exploit sederhana atau otomatisasi scanning.



Active Directory Basics: Memahami cara kerja jaringan perusahaan (Windows Domain) dan kerentanan umumnya.



🏁 Milestone Project: "Capture The Flag (CTF) Challenge" — Selesaikan 5 mesin level Intermediate di platform seperti HackTheBox atau TryHackMe.



🟥 Level 4: The Cyber Sentinel (PRO)

Fokus: Spesialisasi, Red Teaming, dan pertahanan tingkat tinggi.



Red Teaming Operations: Simulasi serangan nyata, evading Antivirus/EDR, dan teknik Post-Exploitation tingkat lanjut.



Malware Analysis \& Reverse Engineering: Membongkar file .exe atau script jahat untuk melihat cara kerjanya menggunakan Ghidra atau x64dbg.



Incident Response \& Forensics: Belajar apa yang harus dilakukan saat terjadi peretasan dan cara melacak jejak penyerang.



Cloud Security: Mengamankan infrastruktur di AWS, Azure, atau Google Cloud.



DevSecOps: Mengintegrasikan keamanan ke dalam alur kerja pengembangan software (CI/CD).



🏁 Milestone Project: "The Final Audit" — Lakukan Full Penetration Testing pada sebuah ekosistem mini (Web, Network, \& Database) dan berikan solusi mitigasinya.

Gamifikasi: Ikut kompetisi CTF (Capture The Flag). Rasanya seperti main game puzzle tapi dengan skill nyata.



Bangun Lab Sendiri: Buat jaringan kecil di laptopmu (pake VMware/VirtualBox). Hubungkan 1 mesin penyerang (Kali/Parrot) dan 2-3 mesin korban.



Join Community: Diskusi dengan sesama pelajar di Discord atau forum lokal. Belajar bareng jauh lebih seru daripada sendirian di kamar.

\# 🛠️ Level 1.0: Linux Fundamental - The Architect's Foundation



Selamat datang di langkah pertama perjalananmu menjadi seorang Cyber Security Professional! Di tahap ini, kita tidak akan langsung meretas. Kita akan membangun pondasi yang sangat kuat di \*\*Linux\*\*.



Kenapa Linux? Karena hampir semua server di dunia, perangkat IoT, dan \*hacking tools\* berjalan di atas Linux. Tanpa menguasai ini, kamu akan kesulitan di level berikutnya.



\---



\## 1. Persiapan: Instalasi Lab

Sebelum mulai, kamu butuh lingkungan untuk praktik. Jangan instal sebagai OS utama dulu jika masih ragu.



\### Pilihan Metode:

1\.  \*\*Virtualization (Rekomendasi):\*\* Gunakan \*\*VirtualBox\*\* atau \*\*VMware Player\*\*. Instal distro seperti \*\*Ubuntu\*\* atau \*\*Linux Mint\*\*.

2\.  \*\*WSL2 (Windows Subsystem for Linux):\*\* Jika menggunakan Windows 10/11, kamu bisa instal Ubuntu langsung dari Microsoft Store.

3\.  \*\*Live USB:\*\* Menjalankan Linux lewat Flashdisk tanpa mengganggu hardisk.



\---



\## 2. Mengenal Struktur Folder (Filesystem Hierarchy)

Berbeda dengan Windows yang menggunakan `C:\\`, Linux menggunakan struktur pohon dimulai dari root `/`.



| Direktori | Fungsi Utama |

| :--- | :--- |

| `/` | \*\*Root:\*\* Puncak dari semua folder. |

| `/bin` \& `/sbin` | Berisi perintah dasar sistem (seperti `ls`, `cp`). |

| `/etc` | \*\*Penting:\*\* Tempat menyimpan file konfigurasi sistem. |

| `/home` | Folder personal untuk user (dokumen, download, dll). |

| `/var/log` | \*\*Krusial untuk Security:\*\* Berisi catatan (log) aktivitas sistem. |

| `/tmp` | File sementara yang akan dihapus saat restart. |



\---



\## 3. Navigasi Terminal (CLI)

Buka terminalmu (`Ctrl + Alt + T`). Ini adalah senjata utamamu.



\### Perintah Dasar Navigasi:

\* `pwd` (Print Working Directory): Cek di mana posisi folder kamu sekarang.

\* `ls`: List file. Gunakan `ls -la` untuk melihat file tersembunyi dan detail izinnya.

\* `cd \[nama\_folder]`: Masuk ke folder.

\* `cd ..`: Kembali ke satu folder sebelumnya.

\* `clear`: Membersihkan layar terminal.



\---



\## 4. Manipulasi File \& Folder

Belajarlah mengelola file secepat kilat tanpa mouse.



\### Operasi Dasar:

\* \*\*Membuat Folder:\*\* `mkdir nama\_folder`

\* \*\*Membuat File Kosong:\*\* `touch nama\_file.txt`

\* \*\*Menyalin:\*\* `cp file\_asal tujuan` (Gunakan `-r` untuk folder).

\* \*\*Memindahkan/Ganti Nama:\*\* `mv file\_lama nama\_baru`

\* \*\*Menghapus:\*\* `rm nama\_file` (Gunakan `rm -rf` untuk menghapus folder secara paksa - \*\*Hati-hati!\*\*).



\---



\## 5. Membaca \& Mencari Isi File

Di Cyber Security, kamu akan sering membaca file konfigurasi atau file log yang panjang.



\* `cat \[file]`: Menampilkan seluruh isi file.

\* `head` / `tail`: Melihat 10 baris pertama atau terakhir sebuah file.

\* `less`: Membaca file secara interaktif (bisa di-\*scroll\*).

\* `grep "kata\_kunci" file`: Mencari kata tertentu di dalam file. Ini sangat berguna untuk mencari celah atau pesan error.



\---



\## 6. Redirection \& Piping

Menghubungkan satu perintah dengan perintah lainnya.



\* `>` : Menyimpan output perintah ke file (menimpa isi).

\* `>>` : Menambahkan output ke akhir file (tanpa menghapus isi lama).

\* `|` (Pipe): Mengirim hasil perintah pertama ke perintah kedua.

&#x20;   \* \*Contoh:\* `ls -la | grep "secret"` (Mencari file yang ada kata "secret" di folder saat ini).
