\## 1. Persiapan: Instalasi Lab

Sebelum mulai, kamu butuh lingkungan untuk praktik. Jangan instal sebagai OS utama dulu jika masih ragu.



\### Pilihan Metode:

1\.  \*\*Virtualization (Rekomendasi):\*\* Gunakan \*\*VirtualBox\*\* atau \*\*VMware Player\*\*. Instal distro seperti \*\*Ubuntu\*\* atau \*\*Linux Mint\*\*.

2\.  \*\*WSL2 (Windows Subsystem for Linux):\*\* Jika menggunakan Windows 10/11, kamu bisa instal Ubuntu langsung dari Microsoft Store.

3\.  \*\*Live USB:\*\* Menjalankan Linux lewat Flashdisk tanpa mengganggu hardisk.



\---



\## 2. Mengenal Struktur Folder (Filesystem Hierarchy)

Berbeda dengan Windows yang menggunakan `C:\\`, Linux menggunakan struktur pohon dimulai dari root `/`.



| Direktori | Fungsi Utama |

| :--- | :--- |

| `/` | \*\*Root:\*\* Puncak dari semua folder. |

| `/bin` \& `/sbin` | Berisi perintah dasar sistem (seperti `ls`, `cp`). |

| `/etc` | \*\*Penting:\*\* Tempat menyimpan file konfigurasi sistem. |

| `/home` | Folder personal untuk user (dokumen, download, dll). |

| `/var/log` | \*\*Krusial untuk Security:\*\* Berisi catatan (log) aktivitas sistem. |

| `/tmp` | File sementara yang akan dihapus saat restart. |



\---



\## 3. Navigasi Terminal (CLI)

Buka terminalmu (`Ctrl + Alt + T`). Ini adalah senjata utamamu.



\### Perintah Dasar Navigasi:

\* `pwd` (Print Working Directory): Cek di mana posisi folder kamu sekarang.

\* `ls`: List file. Gunakan `ls -la` untuk melihat file tersembunyi dan detail izinnya.

\* `cd \[nama\_folder]`: Masuk ke folder.

\* `cd ..`: Kembali ke satu folder sebelumnya.

\* `clear`: Membersihkan layar terminal.



\---



\## 4. Manipulasi File \& Folder

Belajarlah mengelola file secepat kilat tanpa mouse.



\### Operasi Dasar:

\* \*\*Membuat Folder:\*\* `mkdir nama\_folder`

\* \*\*Membuat File Kosong:\*\* `touch nama\_file.txt`

\* \*\*Menyalin:\*\* `cp file\_asal tujuan` (Gunakan `-r` untuk folder).

\* \*\*Memindahkan/Ganti Nama:\*\* `mv file\_lama nama\_baru`

\* \*\*Menghapus:\*\* `rm nama\_file` (Gunakan `rm -rf` untuk menghapus folder secara paksa - \*\*Hati-hati!\*\*).



\---



\## 5. Membaca \& Mencari Isi File

Di Cyber Security, kamu akan sering membaca file konfigurasi atau file log yang panjang.



\* `cat \[file]`: Menampilkan seluruh isi file.

\* `head` / `tail`: Melihat 10 baris pertama atau terakhir sebuah file.

\* `less`: Membaca file secara interaktif (bisa di-\*scroll\*).

\* `grep "kata\_kunci" file`: Mencari kata tertentu di dalam file. Ini sangat berguna untuk mencari celah atau pesan error.



\---



\## 6. Redirection \& Piping

Menghubungkan satu perintah dengan perintah lainnya.



\* `>` : Menyimpan output perintah ke file (menimpa isi).

\* `>>` : Menambahkan output ke akhir file (tanpa menghapus isi lama).

\* `|` (Pipe): Mengirim hasil perintah pertama ke perintah kedua.

&#x20;   \* \*Contoh:\* `ls -la | grep "secret"` (Mencari file yang ada kata "secret" di folder saat ini).
